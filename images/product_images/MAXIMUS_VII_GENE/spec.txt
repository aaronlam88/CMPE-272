Note
	*1: The PCIe 2.0 x4 slot (PCIEX4_1) shares bandwith with mPCIe 2.0 x1 and M.2 slots. The default setting is in Auto mode that automatically optimizes the system bandwidth. If you install PCIe 2.0 x4 device, the system will automatically detect and disable mPCIe 2.0 x1 and M.2 slots.
	*2: The supported functions depends on the CPU installed.
	*3: These SATA ports are for data hard drives only. ATAPI devices are not supported.
	*4: 8-channel audio output is supported by jack-retasking using a chassis with HD audio module in the front panel.
	*5: Support ASUS USB 3.0 Boost, UASP standard on the Intel native USB 3.0 is only supported under Windows® 8.1/8.
	*6: 1 x USB2.0 ports at mid-board (USB13) shares with ROG extension (ROG_EXT) port.
